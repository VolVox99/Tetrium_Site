Version 1.1

New gameplay customizations, quality of life features, bug fixes, and more

23
NOV

Lots o' improvements!

  • Features
    • Added ability to adjust ARR, DAS, and Soft Drop speed
    • Your Discord Presence will now update according to what you are doing in the game
    • Ability to adjust master, music, and sound effects volume
    • Multiple soundtracks to listen to
    • Added ability to resize the window or make it fullscreen (this one was a pain)
    • All problems will be logged so we can quickly identify bugs for future updates
    • Made it easier to block garbage in your own junk meter
    • Added more themes
    • Added a random theme option
    • Nickname now saves, and is automatically entered upon startup
    • Updated block appearance to look nicer
  • Bug Fixes
    • Fixed issue which caused the game to crash occasionally
    • Fixed bug where S and Z pieces spawned upside down
    • Got rid of the quick flash of the game before the countdown
    • Fixed issue where holding soft drop before a rematch would cause the piece to go extra fast the next round
    • Speed level announcements and the end screen now matches the theme
    • Corrected problem where some T-Spin minis were being counted as regular T-Spins
    • Fixed scoring system so it multiplies by the speed level rather than the height level
    • No longer takes forever to open the game
